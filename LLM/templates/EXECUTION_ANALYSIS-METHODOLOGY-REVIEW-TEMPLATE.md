## üìö Usage Guidelines

**When to Use This Template**:
- PLAN completion (at END_POINT protocol)
- Periodic methodology review
- Methodology gap discovered
- Compliance audit needed
- Extracting learnings from completed work

**How to Fill This Template**:
1. **Header**: Fill in Purpose, Date, Status, Related PLAN(s), Category
2. **Executive Summary**: High-level findings and recommendations
3. **Findings by Category**: Organize findings (what worked, what didn't, metrics)
4. **Recommendations**: Methodology improvements, template updates, protocol enhancements
5. **Action Items**: Specific improvements to implement (immediate, short-term, long-term)
6. **Conclusion**: Summary and next steps

**Best Practices**:
- Be evidence-based (include metrics and examples)
- Organize findings by category for clarity
- Prioritize action items (immediate, short-term, long-term)
- Link to related analyses and PLANs
- Update status as review progresses
